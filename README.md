# Gamified Learning Progress Tracker

React-based Single Page Application for tracking academic learning progress with XP, badges, streaks, charts, and a leaderboard.

## Features

- Login with name and roll number
- Protected routes using React Router
- Per-user LocalStorage progress
- Dashboard with XP, level, streak, and subject cards
- Manual topic completion system
- Badge gallery with locked and unlocked states
- Recharts line chart for XP history
- Recharts bar chart for subject progress
- Leaderboard ranked by XP
- Profile settings with avatar color and reset progress
- Dark mode toggle saved to LocalStorage
- Express mock API for motivational quotes
- Postman collection for mock API testing

## Run Locally

Install dependencies:

```bash
npm install
```

Start the mock API:

```bash
npm run server
```

Start the React app in another terminal:

```bash
npm run dev
```

Open:

```text
http://127.0.0.1:5173
```

## Build

```bash
npm run build
```
